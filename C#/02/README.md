# C# for문 학습노트
for문이란?

정해진 횟수만큼 반복할 때 사용하는 반복문

반복할 횟수를 초기값, 조건식, 증감식으로 설정함

기본 구조
```for (초기값; 조건식; 증감식)
{
    // 반복 실행할 코드
}
```
구조 설명
초기값: 반복을 시작할 변수 설정 → int i = 0
조건식: 이게 true인 동안 계속 반복 → i < 10
증감식: 반복할 때마다 변화 → i++ (1씩 증가)

간단한 예제
```
for (int i = 1; i <= 5; i++)
{
    Console.WriteLine(i);
}
```
```
1
2
3
4
5
```
Tip
i++ 대신 i += 2 쓰면 2씩 증가 가능

break로 반복 중간에 종료 가능

continue로 특정 조건일 때 건너뛰기 가능
